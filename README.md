ESP32-Based Mains-Isolated Clap-Activated Relay Switch

Custom KiCad PCB integrating an analog microphone front-end, ESP32-C6 processing, and a transistor-driven relay stage to safely switch a 120VAC load.

Includes schematic, PCB layout, BOM, and custom symbols/footprints.

⚠ This project involves mains voltage and should be handled with proper safety precautions.
